Overview

This is a fully featured 3D space exploration and combat game built from scratch in Unity using C#. The project covers a wide range of game development disciplines — from rendering and physics to AI behavior and economy systems — all built and integrated by hand.

Core Features

🪐
Planetary Landing
Seamless transition from space flight to planetary surface with full terrain rendering.
☁️
Volumetric Clouds
Real-time volumetric cloud rendering using ray-marching shaders for immersive atmosphere.
🤖
AI Enemy Ships
Enemy ships with behavior trees — patrol, detect, intercept, and attack using state machine AI.
🚀
Ship Economy
In-game economy allowing players to earn credits and purchase new ships with different stats.

Technical Highlights

The volumetric clouds are implemented via a custom ray-marching shader in HLSL — sampling a 3D noise texture at multiple steps to simulate light scattering through cloud density. This runs in real time and reacts to the game's lighting system.

Enemy AI uses a finite state machine with states for idle patrol, threat detection, pursuit, attack, and retreat. Each ship independently evaluates its situation each frame and transitions states based on proximity, health, and line-of-sight checks.

Planetary landing uses a scale transition system — the game seamlessly blends between the space camera and the surface camera as altitude decreases, without a loading screen.
