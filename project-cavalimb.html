Project Overview

CavaLimb is a student-led project at UVA developing a functional prosthetic arm from the ground up. The team is interdisciplinary, spanning mechanical engineering, electrical engineering, and computer science, with the shared goal of producing a capable, accessible prosthetic device.

My Role

I'm responsible for the finger and hand movement subsystem — the most mechanically complex part of the arm. This involves designing the linkage geometry and actuation mechanism that translates inputs (from sensors or a control interface) into natural, precise articulation of each finger.

The challenge is balancing mechanical simplicity, grip strength, range of motion, and reliability — each finger needs to move naturally with minimal bulk and weight.

Technical Approach

The current design uses a tendon-driven actuation system — cables routed along each finger that are pulled by small servo motors in the forearm. This mirrors the anatomy of a real hand and allows for a compact, lightweight design. Each finger can flex and extend independently, enabling both power grips and precision pinch actions.

CAD work is done in Fusion 360, with prototypes being fabricated and tested iteratively. Embedded programming for the servo control system is written in C++ for Arduino-compatible hardware.
