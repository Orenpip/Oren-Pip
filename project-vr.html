Published Research · Journal for Emerging Investigators · 2025

AI Face-Tracking
VR Gaming

A real-time computer vision system that translates facial movements into in-game interactions, creating an immersive semi-virtual reality experience. Published in the Journal for Emerging Investigators.

Published Computer Vision Unity / C#

Concept

This project explores a novel interaction paradigm for gaming: using a standard webcam and real-time AI face-tracking to control in-game actions through facial expressions and head movement — no controllers required. The result is a semi-virtual reality experience that bridges the physical and digital worlds without expensive VR hardware.

Technical Approach

The system uses a computer vision face-tracking pipeline built in Python that processes webcam frames in real-time to extract facial landmark positions and movement vectors. Key tracked elements include head tilt and pan, mouth open/close state, eyebrow raise, and eye movement direction.

These signals are mapped to in-game actions inside a Unity game environment built in C#. A custom bridge layer handles the real-time communication between the Python CV backend and the Unity game engine, keeping latency low enough for responsive gameplay.

Gameplay Integration

The game was designed specifically around the face-tracking interface — mechanics that would feel natural to control with facial gestures rather than shoehorned from a traditional controller scheme. The result is a genuinely immersive experience where the player's face is the controller.

Publication

The project was written up and submitted to the Journal for Emerging Investigators, a peer-reviewed publication for pre-college and undergraduate researchers. It was accepted and published in December 2025.
